About

No BS Free power and speed material test grid generator service.

The idea for this project came when I got my first laser. It really did take some time to create a material and speed test and as much as this can help you understand how your machine works creating one takes time and results are not always satisfactory if you don't invest the time. As a software developer it is natural for me to automate complex tasks which usually results in more time for creativity and better quality end results.

I'm sharing this tool with you. It is free to use, and I'll try to keep it that way! If you like it, buy me a beer! It will help support my family, keep this site running and create more cool features for it.

P.S: I would love to see some pictures of the engraved/cut grids you generate with it. So please send some via mail or share on the facebook page!
